In the State of Takahashi in the AtCoderian Federation, there are N cities numbered from 1 to N, connected by M bidirectional roads. Each road connects two distinct cities. The State of Takahashi is to be divided into two states, Taka and Hashi, and every city must belong to either Taka or Hashi, with the requirement that any two cities within the same state must be directly connected by a road. Your task is to find the minimum possible number of roads that have endpoints in the same state. If it isn't possible to divide the cities in a satisfactory manner, return -1.

Unit Converter: Given a number with an optional prefix and unit, convert it into scientific notation with the same significant figures. The prefixes include yotta, zetta, exa, peta, tera, giga, mega, kilo, hecto, deca, deci, centi, milli, micro, nano, pico, femto, ato, zepto, yocto. The input will consist of a number (which may contain a decimal point), an optional prefix, and a unit, all separated by spaces. You must output the quantity in the format a * 10^b [unit] where 1 <= a < 10. The singular or plural form of the unit name does not affect the output.

The problem involves creating a mobile, which is a type of moving art piece, using rods, strings, and weights. The task is to ensure that the mobile is balanced and to find the configuration of weights that minimizes the total weight of the mobile while maintaining balance. Each rod has a specified ratio for the lengths from the pivot to the red and blue ends, and it may have other rods or weights attached to it. The program should output the minimum total weight of the mobile for each data set provided.

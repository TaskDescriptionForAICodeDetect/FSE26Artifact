You are the director of a concert hall facing bankruptcy. The hall has two identical rooms that can each host one concert per day. Applicants request a room for a specified period [i, j] and offer a price w they are willing to pay for the entire period. Your goal is to select applications to maximize the total income for the year, considering that each application can either be fully accepted or completely rejected. The input contains multiple datasets with the number of applications followed by the details of each application. The output must show the maximum total income for each dataset.
